## pyCode
代码功能为输入txt文件的长度不限的声音信号和震动信号（需同步），以1024点为一帧分别使用预训练模型对信号进行分析，输出两模型基础概率分布作为证据理论的基础概率指派。

声音信号为提取MFCC特征进行识别，震动信号为直接将信号喂入。

##cCode
cCode为该算法的C语言实现。
